# Genghis-Capital
This is a demo code to get a value missing in an array of integer.

## Installation 
### Step 1
- Install python 3.8 and clone the repository.
- cd /workdir
- Run python ./main.py
  
### Step 2
- Install docker.
- docker run --rm -p 5000:5000 devtechguy/genghis-capital
- check on the port you choose to run on.
